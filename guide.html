iOS
iPhone / iPad 安裝教學
iOS 只能透過 Safari 安裝 PWA，使用 Chrome 或 LINE 內建瀏覽器無法安裝。
1️⃣

用 Safari 開啟網站

打開 Safari 瀏覽器，在網址列輸入試衣間的網址，或掃描 QR Code。

2️⃣

點底部的「分享」按鈕

點擊 Safari 底部工具列中間的 ⬆️ 分享按鈕（一個方框加上向上箭頭的圖示）。

3️⃣

選「加入主畫面」

在分享選單中往下滑，找到 「加入主畫面」（Add to Home Screen），點擊它。

4️⃣

確認名稱，點「新增」

名稱可以保留預設「虛擬試衣間」或自己改，然後點右上角的 「新增」。

✅

完成！

桌面會出現試衣間的圖示，點擊即可全螢幕開啟，使用體驗跟原生 APP 一樣。

安裝後開啟不會顯示 Safari 的網址列和工具列，看起來就像真正的 APP。
ANDROID
Android 手機安裝教學
1

用 Chrome 開啟網站

打開 Chrome 瀏覽器，輸入試衣間網址。

2

點右上角 ⋮ 選單

點擊 Chrome 右上角的三個點選單。

3

選「安裝應用程式」或「加到主畫面」

選單中找到「安裝應用程式」或「加到主畫面」，點擊後確認即可。

Android 的 Chrome 通常會在網址列下方自動彈出「新增到主畫面」的提示，點擊即可快速安裝。
STEP 02
開始使用
1

Google 登入

首次開啟會看到登入畫面，點「使用 Google 登入」。登入後衣櫃和設定會自動同步到雲端，換手機也不會遺失。

2

設定 API 金鑰

點右上角「設定」，輸入你的 Replicate API 金鑰。到 replicate.com/account/api-tokens 註冊並取得金鑰。

3

上傳個人照片

在主畫面點擊人形圖示，選一張清楚的全身照或半身照作為試穿模特兒。

照片建議：正面面對鏡頭、光線充足、背景簡單的全身照效果最好。
STEP 03
管理衣櫃

衣櫃是你所有衣物的收藏庫，分為三個類別：

👔

上衣

T恤、襯衫、外套、衛衣等

👖

下身

牛仔褲、裙子、短褲、長裙等

👗

洋裝

連身裙、洋裝、連身褲等

＋

新增衣物

點「衣櫃」→ 選擇類別 → 點「+ 上傳」→ 選擇衣物照片。建議使用商品圖（白底或乾淨背景），效果最佳。

✎

編輯衣物

在衣櫃頁面點右上角「編輯」，可以刪除不需要的衣物。

✓

選擇衣物

在衣櫃或主畫面底部點擊衣物即可選取，再點一次取消。右側會顯示已選衣物，點擊也可取消。

預設已有示範衣物供試用，你可以隨時上傳自己的衣物替換。
STEP 04
虛擬試穿
1

選擇衣物

從底部衣物列選取想試穿的衣物。可以同時選擇上衣＋下身，系統會依序試穿。

2

點「試穿」

確認已上傳個人照片且選好衣物後，點擊白色的「試穿」按鈕。

3

等待生成

AI 會將衣物穿到你的照片上，每件衣物約需 15-30 秒。畫面會顯示進度。

4

檢視與下載

完成後可以預覽試穿結果，點「下載」儲存圖片。

每次試穿消耗約 NT$0.8，費用從你的 Replicate 帳戶扣除。
STEP 05
購物助手

逛街或網購時看到喜歡的衣服，拍照上傳讓 AI 根據你的衣櫃分析值不值得買。

1

上傳商品照片

在主畫面點「購物助手」→「+ 上傳商品照片」。

2

AI 分析

點「分析」，AI 會根據你衣櫃裡的衣物給出建議：值得購入或建議放棄，以及搭配方案和風格點評。

FAQ
常見問題
試穿需要多少錢？
每件衣物試穿約 NT$0.8（約 USD$0.025），從你的 Replicate 帳戶餘額扣除。新帳戶通常有免費額度可以試用。
什麼樣的照片效果最好？
個人照片：正面、全身、光線充足、背景簡單。衣物照片：商品圖（白底）效果最好，真人穿著的照片也可以。
我的照片安全嗎？
照片透過 Replicate API 處理，不會儲存在我們的伺服器。API 金鑰只存在你自己的瀏覽器和 Firebase 帳戶中。
為什麼只有上衣、下身和洋裝？
目前的 AI 模型（IDM-VTON）專門針對衣物試穿優化，對帽子、鞋子、包包等配件的效果不理想，因此暫時只提供效果穩定的三個類別。
可以離線使用嗎？
衣櫃瀏覽可以離線使用（資料有本地快取），但試穿功能需要網路連線來呼叫 AI 模型。
換手機怎麼辦？
只要用同一個 Google 帳號登入，衣櫃和設定會自動從雲端同步回來。
iOS 上無法登入 Google？
請確認使用 Safari 開啟（不是 LINE 或 Facebook 內建瀏覽器），並確認網域已加入 Firebase 授權名單。
